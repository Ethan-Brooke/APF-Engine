# W_TRACE release attestation template pack

This directory contains a **template-only** release attestation shape for the W_TRACE release pipeline. It is not a real signed attestation and cannot unlock physical W export.

A future real release attestation must bind a validated release packet to an immutable manifest digest and a real detached signature or signature bundle. Observed physical `M_W`, world-average W mass, APF-anchor backsolves, residual fits, and manual export overrides remain forbidden inputs.
